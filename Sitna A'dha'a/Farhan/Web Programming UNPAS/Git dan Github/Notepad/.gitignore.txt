.gitignore

- jika berbentuk file maka ditulis original
 cth :
 - config.txt
- jika berbentuk folder juga ditulis original namun dengan slash (/) diujungnya
 cth :
 - data/
- jika berbentuk pola
 cth :
 - *.exe (semua file yang berekstensi exe)
